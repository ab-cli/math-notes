# MATH2352 Interactive Lecture Notes

AI-generated interactive notes for Differential Equations (MATH2352). Each page includes theory, worked examples, and practice questions with step-by-step solutions.

**[Live Demo](https://ab-cli.github.io/math-notes/)** — browse the notes in your browser

## Topics Covered

### First Order ODEs (Weeks 1–2)
- What are Differential Equations — classification, order, direction fields
- Partial Derivatives (supplementary review)
- Modelling — population growth, mixing problems, Newton's cooling
- Integrating Factors
- Separation of Variables
- Euler's Numerical Method
- Exact and Non-exact Equations

### Second Order Linear ODEs (Weeks 3–4)
- Homogeneous Equations — characteristic equation, real/complex roots
- Wronskians and Linear Independence
- Repeated Characteristic Roots and Reduction of Order
- Non-homogeneous Equations — undetermined coefficients
- Variation of Parameters

## Features

- Interactive exercises with instant feedback
- Multi-step guided solutions (MCQ + worked problems)
- MathJax-rendered equations
- Sidebar navigation within each lecture
- Dark theme, mobile-responsive

## Tech

Each lecture is a self-contained HTML file with inline CSS and JavaScript. Math rendering via [MathJax 3](https://www.mathjax.org/).

No build step, no dependencies — just open any `.html` file in a browser.
